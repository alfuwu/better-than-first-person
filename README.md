This is a port of [First-person Model](https://modrinth.com/mod/first-person-model) to Better than Adventure. There is no configuration available for this version, nor keybind, but you're welcome to add both of those features to the mod yourself if you feel like it by [opening a pull request on the mod's GitHub repo](https://github.com/alfuwu/better-than-first-person).

This mod was not exhaustively tested, so if you find any issues, please make sure to report them on [the issue tracker](https://github.com/alfuwu/better-than-first-person/issues) so it can get fixed!
